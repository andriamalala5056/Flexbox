Flexbox

Définition :

FlexBox (pourFlexible Box) est un mode de mise en page prévoyant une disposition des éléments d'une page de telle sorte que ces éléments possèdent un comportement prévisible lorsqu'ils doivent s'accommoder de différentes tailles d'écrans/appareils.

Source : developer.mozilla.org

1. Display:flex :

L'affichage ci-dessous montre trois blocs dans un conteneur, ici le containeur n'est pas encore en mode "flex".

Voyons maintenant comment est la disposition des blocs après display:flex de notre containeur

.container {
    display: flex;
}
		
<div class="container">
    <div class="element-1"></div>
    <div class="element-2"></div>
    <div class="element-3"></div>
</div>

2. Flex-direction :

Pour flex-direction, plusieurs paramètres sont disponibles...
Voici les listes :

  • row (affiche les éléments sur une ligne)
  • column (affiche les éléments en colonne)
  • row-reverse (affichage en ligne mais renversé)
  • column-reverse (affichage en colonne mais renversé)

.container {
    display: flex;
    flex-direction: row-reverse;
}

Les autres paramètres pour flex-direction s'utilisent de la même manière. Voici un autre exemple pour "column-reverse"

.container {
    display: flex;
    flex-direction: column-reverse;
}

Nous pouvons constater que le jaune et le bleu se sont échangés de place.

3. Flex-wrap :

Flex-wrap permet de mettre à la ligne les éléments automatiquement quand ces derniers n'on plus d'espace disponible.
Nous avons 3 paramètres pour flex-wrap

  • wrap
  • wrap-reverse
  • nowrap

Prenons l'exemple de row-reverse, si on reduit le navigateur les éléments se rétrecient et se collent entre eux, mais l'exemple suivant vous montre le contraire

.container {
    display: flex;
    flex-direction: row-reverse;
    wrap: wrap;
}

L'élément mise à la ligne se met à droite car on a pris l'exemple pour row-reverse. Voyons un autre exemple sans le "row-reverse".

.container {
    display: flex;
    wrap: wrap;
}

4. Justify-content :

Justify-content permet d'aligner les éléments dans le conteneur horizontalement.
Nous avons :

  • flex-start
  • flex-end
  • space-between
  • space-around
  • center

Voyons un exemple pour "space-between"

.container {
    display: flex;
    justify-content: space-between;
}

Les éléments s'étendent sur toute la largeur du conteneur avec des espacements égaux, même cas pour "space-around" sauf qu'il ajoute une marge à droite et à gauche pour l'ensemble des éléments.

5. Align-items :

Align-items permet d'aligner l'ensemble des éléments verticalement.

  • flex-start
  • flex-end
  • center
.container {
    display: flex;
    align-items: center;
    height: 400px; /* Juste pour qu'on puisse voir que les éléments sont bien centrés verticalement */
}

Si vous voulez placer les éléments au centre du conteneur, c'est à dire verticalement et horizontalement, il suffit juste d'ajouter "justify-content:center" à ce code, en voici un exemple :

.container {
    display: flex;
    align-items: center;
    justify-content: center;
    height: 400px; /* Juste pour qu'on puisse voir que les éléments sont bien centrés verticalement */
}

6. Align-self :

L'align-self s'applique aux éléments dans le conteneur, c'est à dire que le css selecteur css cible les éléments.

  • flex-start
  • flex-end
  • center
1
2
3
.container {
    display: flex;
    align-items: center;
}

.element-1 {
    align-self: flex-start;
}
